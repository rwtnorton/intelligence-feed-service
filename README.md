# intelligence-feed-service

REST microservice that provides search capabilities on AlienVault OTX intelligence feed.

## Usage

Build uberjar and run locally:
```
$ make uber run
```

Build and run in Docker:
```
$ make uber docker-build docker-run
```

### Linting

```
$ make lint
```

### Formatting

```
$ make cljfmt-check
```

```
$ make cljfmt-fix
```

### Testing

```
$ make test
```
